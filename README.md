# SOBERIMUSOR

**Управление:**
- G - Экипировать/Убрать оружие
- R - Перезарядить оружие
- Tab - Открыть/Закрыть оружие
- V - Открыть/Закрыть редактор робота
- E - Взаимодействовать с объектом
- F - Подобрать предмет
